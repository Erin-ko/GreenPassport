## 1. 核心技術選型 (Tech Stack)
- **前端**：純 HTML5/CSS3/JS (Vanilla MPA，多頁面 RWD 網頁)。
- **後端**：FastAPI。
- **資料庫**：SQLite (本地開發環境) / Render PostgreSQL (正式部署環境)。
- **部署方式**：前端 `frontend/` 目錄直接交由後端 FastAPI 託管，於 Render.com 部署單一 Web Service 免費方案，免去前端打包步驟，啟動更迅速。

## 2. 業務規則與限制條件 (Business Rules & Constraints)
- **3天完成 Demo 限制**：優先實現核心 MVP（護照增刪查改、保養提醒、微型互助看板），第二階段之 ESG 永續統計、動態進度條等功能在此版本暫不實作。
- **圖片限制 (500KB)**：為防範免費 Render PostgreSQL (1GB) 容量迅速耗盡，上傳之圖片必須限制在 500KB 以下，且以 Base64 字串形式存放於資料庫中。
- **提醒決策**：因 Render 免費伺服器會休眠，Demo 階段的主動提醒不使用背景 Cron 發信，改為**「使用者登入或開啟系統時，在前端畫面跳出即期物品提醒」**。
- **隱私地理計算**：互助看板需使用約略定位（保留兩位小數之經緯度，精確度約 1 公里），後端計算 Haversine 距離，API 僅向前端返回模糊距離文字（例如：「距離約 300m」），不得暴露精確座標。
